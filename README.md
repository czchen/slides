# Slides

This project stores slides from czchen.

# How to use

*   Install [markdown-to-slides](https://www.npmjs.com/package/markdown-to-slides) from npm.
    ```sh
    npm install -g markdown-to-slides
    ```

*   Use [markdown-to-slides](https://www.npmjs.com/package/markdown-to-slides).
    ```sh
    markdown-to-slides -d my-slide.md -o my-slide.html
    ```
